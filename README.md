# AMAÇ

Uçtan uca plaka tespiti yapan bir model geliştirmek. Bu modeli başlıca 2 bölüme ayırdık: nesne tespiti ve optik karakter okumak. 

# GENEL AKIŞ

1- Plaka tespiti

2- Karakter Tespiti

3- Karakter Tanıma

4- Optimizasyon


# DENENEN YÖNTEMLER

- Derin öğrenme kullanılarak tespit edilecek plakanın bounding box köşeleri tahmin edilmeye çalışıldı. Yani probleme bir regresyon problemi olarak yaklaşıldı.
